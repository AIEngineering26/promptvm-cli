# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the PromptVM CLI, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please use [GitHub Security Advisories](https://github.com/AIEngineering26/promptvm-cli/security/advisories/new) to report the vulnerability privately.

Alternatively, email **security@promptvm.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

This policy covers the `promptvm` CLI binary and its source code. For vulnerabilities in the PromptVM platform (API, web app, MCP server), please report them separately at https://github.com/AIEngineering26/promptvm/security.
